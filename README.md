Flatzo is a simple and user-friendly flat booking web application designed to help users explore and book residential flats online with ease. The project focuses on providing a smooth browsing experience where users can view different types of flats, compare details, and confirm their choice in just a few clicks.

The application showcases multiple flat categories such as 2BHK, 3BHK, and 4BHK, each with detailed information including price, location, images, and descriptions. Flatzo uses a clean UI design and intuitive navigation to ensure users can move seamlessly between the home page, flat listing page, and individual flat detail pages.

This project is developed using HTML, CSS, Bootstrap, and basic JavaScript, emphasizing front-end development concepts like responsive design, component-based layouts, and interactive page transitions. Flatzo demonstrates how real-world property booking platforms work at a basic level and serves as a strong foundation for learning modern web development.

Key Features

Attractive home page with call-to-action

Flat listing with multiple BHK options

Detailed flat description pages

Price and location display

Simple booking confirmation flow

Responsive and visually appealing UI
